# frameworks-study
Попытка осмыслить принцип работы JavaScript фреймворков на нативном JS. Просто эксперименты и ничего более 🙃🙃🙃

## Установка
> `npm i`

## Скрипты
### Запуск для разработки
> `npm start`
### Сборка
> `npm run build`
### Сборка документации
> `npm run docs:build`
